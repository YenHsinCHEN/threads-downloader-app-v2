# Threads Downloader App v2

增強版 Threads 影片和圖片下載器 for Android

## 📱 新功能
- ✨ 支援圖片下載
- 🎥 支援影片下載
- 📝 顯示作者名稱和貼文內容
- 🔄 改進的重置功能
- 🎯 更準確的媒體偵測
- 📊 優化的使用者介面

## 🚀 使用方法
1. 開啟應用程式
2. 瀏覽 Threads 網站
3. 點擊左側搜尋按鈕偵測媒體
4. 選擇要下載的影片或圖片
5. 點擊下載按鈕

## 📝 版本歷史
- v2.0 - 新增圖片下載功能、改進偵測邏輯
- v1.0 - [原始版本](https://github.com/YenHsinCHEN/threads-downloader-app)（僅支援影片）

## 🛠️ 技術規格
- Min SDK: 24 (Android 7.0)
- Target SDK: 33
- Language: Kotlin
